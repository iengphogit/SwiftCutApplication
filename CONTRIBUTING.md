# Contributing to SwiftCutApplication

Thank you for your interest in contributing!

## How to Contribute
1. Fork the repository
2. Create a feature branch
3. Commit your changes with clear messages
4. Open a Pull Request

## Guidelines
- Follow Android and iOS best practices
- Avoid restrictive licenses (GPL/AGPL)
- Prefer native media APIs
